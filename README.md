# 🎈 Balloon Popping Game (เกมปาโป่งสุดสนุก)

เกมแนวอาร์เคด (Arcade Game) แบบเรียบง่ายที่พัฒนาด้วยภาษา **Python** และไลบรารี **Pygame** โดยผู้เล่นต้องใช้ความไวของเมาส์ในการคลิกปาเป้าหรือจิ้มลูกโป่งหลากสีที่ลอยขึ้นมาจากขอบจอด้านล่างเพื่อสะสมคะแนนให้ได้มากที่สุด!

---

## 🎮 ฟีเจอร์เด่นของเกม (Game Features)
* **Dynamic Balloon Spawning:** ระบบสุ่มตำแหน่ง ขนาด ความเร็ว และสีสันของลูกโป่งแบบอัตโนมัติ ทำให้เกมมีความท้าทายตลอดเวลา
* **Physics & Boundary Check:** ลูกโป่งจะลอยขึ้นด้านบนด้วยความเร็วที่แตกต่างกัน หากผู้เล่นปล่อยให้ลูกโป่งลอยทะลุขอบจอไปจะถูกนับแต้มพลาด (Missed)
* **Pythagoras Collision Detection:** ใช้หลักการคำนวณระยะทางทางคณิตศาสตร์แบบละเอียด (Pythagorean Theorem) เพื่อตรวจจับว่าพิกัดการคลิกเมาส์ตรงกับพื้นที่วงกลมของลูกโป่งอย่างแม่นยำ
* **Game State Management:** มีระบบหน้าจอ Game Over แสดงคะแนนรวมทั้งหมด และสามารถคลิกเมาส์เพื่อเริ่มเกมใหม่ (Restart) ได้ทันที
